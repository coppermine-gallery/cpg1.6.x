HTML5 Upload Help

This method will allow you to upload a number of files with a single selection action. You will get visual progress feedback for each file as it is being uploaded as well as for the total task. Please take notice of the maximum file size allowed for any single file.


Steps to start upload:

  • Select the album into which you want the uploaded files to go.
  • Select whether you want each item's file name to be used as its title.
  • Enter text in any other displayed fields that will be applied to every file that is uploaded.
  • Select files by clicking on the button to get a file selection dialog or drag and drop files to the specified drop area. Uploading will begin immediately after file selection.
  • While the files are processing, you may add more files to the queue by use of the button or drop area.

When uploading is complete, you will be provided an opportunity to edit the attributes for each uploaded file.
