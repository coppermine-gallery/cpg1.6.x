Téléchargement par HTTP

L'utilisation normale de téléchargement par HTTP utilise les capacités inclues dans le navigateur web de télécharger des fichiers sur un serveur. La taille maximale de fichier est déterminée par deux facteurs de base: la vitesse et le volume de données que le serveur peut télécharger avant le dépassement de temps et la taille maximale de fichier déterminée par le paramétrage du serveur. Notez que ces paramètres ne sont pas déterminés par Coppermine, mais par la configuration du serveur (php.ini). Les utilisateurs qui ont un hébergement web ne peuvent pas modifer le php.ini, ils devront donc faire avec les paramètres définis par l'administrateur du serveur. Ceux qui ont leur propre serveur et peuvent modifier le php.ini doivent regarder ces réglages qui sot expliqués dans la page de résolution des problèmes de téléchargement si les téléchargements par HTTP (les gros fichiers) posent problème.

Il est évident que les fichiers téléchargés doivent aller quelque part (dans un dossier) de votre serveur - c'est le répertoire albums qui se trouve dans le répertoire ou vous avez installé Coppermine sur votre serveur. Les téléchargements par HTTP vont dans un sous-dossier du dossier "userpics" (qui se trouve dans le répertoire "albums"). Evidemment, le script de téléchargement de Coppermine doit avoir les droits d'écriture pour télécharger des fichiers à cet endroit. C'est pourquoi vous devez changer les droits du répertoire albums et de ce qu'il contient pendant l'installation de Coppermine - les rendre inscriptibles pour les utilisateurs du serveur. Ceci est fait en utilisant la commande CHMOD sur un serveur Unix/Linux. Si vous avez des problèmes de téléchargement, assurez vous d'avoir paramétré les droits correctement.

Interfaces de téléchargement

Coppermine comprends deux interfaces de téléchargement: une traditionelle utilisant ue methode entièrement en HTML et permettant de télécharger un seul fichier à la fois et une interface avancée Flash, permettant le téléchargement de plusieurs fichiers en même temps depuis le même écran.

Interface Flash de téléchargement multiples

L'interface Flash offre plus de fonctions et est plus facile d'utilisation, c'est pourquoi elle est recommandée comme interface par défaut.

Utilisation

  1. Sélectionnez un album de la liste déroulante des albums
  2. Cliquez sur le bouton "Parcourir" et allez jusqu'au fichier que vous voulez télécharger et sélectionnez le
  3. Sélectionnez plus de fichiers en répétant l'étape 3
  4. Cliquez sur le bouton "Continuer" une fois que vous avez sélectionné tous les fichiers que vous voulez télécharger (Le bouton apparait une fois que vous avez téléchargé au moins un fichier).
  5. Vous serez dirigé vers une page ou vous pourrez entre les détails concernant les fichiers téléchargés. Après avoir complété ces données, validez le formulaire en utilisant le bouton "Appliquer les modifications" au bas de ce formulaire.

Interface de téléchargement HTML

L'interface de téléchargement HTML est recommandée pour ceux qui n'ont pas Flash d'installé (pour quelue raison que ce soit) et particulièrement pour résoudre les problèmes de téléchargement.

Utilisation

  1. Sélectionnez un album depuis la liste déroulante des albums
  2. Cliquez sur le bouton "Parcourir" et allez jusqu'au fichier que vous voulez télécharger
  3. Entrez le titre du fichier, sa description et les mots clé dans les champs correspondants si possible
  4. Cliquez sur le bouton "Télécharger le fichier" et validez le formulaire pout télécharger le fichier.
