Administration de Coppermine

Comme la plupart des applications qui ont une sorte de réglages pour l'utilisateur, Coppermine comprends des taches qui ne peuvent être faites que par un administrateur. Il est préférable de ne pas nommer le compte administrateur juste "admin" ou "administrateur", mais d'utiliser votre vrai nom ou un pseudo que vous voudrez voir apparaitre à côté de vos téléchargements et de vos commentaires.
Il est indispensable que vous mémorisiez les données de votre compte administrateur et/ou les placiez à un endroit sécurisé; de même que vous pouvez régler votre navigateur pour qu'il mémorise le mot de passe, vous risquez d'en avoir besoin plus tard.
Les comptes qui ont des pouvoirs d'administrateurs ou non sont déterminé par l'appartenance à un groupe d'utilisateur: les membres du groupe "administrators" ont les droits d'administrations. Vous pouvez gérer vos groupes dans le "panneau de gestion des groupes".
Techniquement, il est possible d'avoir plus d'un administrateur pour la galerie, mais il est recommandé d'être prudent avant de donner les droits d'administrations à d'autres: il ne faut donner les pouvoirs d'administrations qu'a des personnes de confiance totale.

Identifié comme administrateur, vous verrez un menu administrateur affichant des options complémentaires et des entrées de menu que vous (en tant qu'admin) avez à comparé aux utilisateurs normaux (non admin) ou aux visiteurs.

Les réglages les plus basiques aussi bien que les réglages avancés ayant un impact sur l'ensemble de votre galerie sont définis dans la page de configuration - certains sont obligatoires pour paramétrer correctement votre galerie lors d'une première utilisation, il est donc recommandé de regarder attentivement la section configuration.

Dans Coppermine, les fichiers sont organisés dans une structure de catégories/albums. Pour paramétrer les catégories et les albums et pour leur maintenance, utilisez le panneau de gestion des catégories and the Album Manager.

Les Bulletin Board Code (connus comme "BBCODE") peuvent être utilisés pour ajouter des liens, des adresses courriel et un formatage basique aux champs textes comme les champs de description.

Votre galerie Coppermine est capable d'extraire les métadonnées des images et de les stocker dans la base de données. Vous pouvez gérer les données au format EXIF en utilisant le gestionnaire d'EXIF.

Des fonctions complémentaires qui ne sont pas nécessaires pour toutes les galeries peuvent être ajoutées en utilisant des plugins.

Coppermine peut appliquer un filigrane (watermark) aux images téléchargées afin de minimiser le vol de contenu.
