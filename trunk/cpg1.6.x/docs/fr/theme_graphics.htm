Ressources graphiques dans les thèmes

Les resources graphiques à l'intérieur des thèmes peuvent être personnalisées et configurées. Les standards suivants sont utilisés pour changer ou modifier l'aspect de votre thème personnalisé:

  • Icones de Menu
  • Barre Graphique de Navigation
  • Images de Notation

Icones de Menu

Le terme icones de menu peut être un peu trompeur: il dépend de l'option de configuration correspondante "Activer les icones de menu". Il n'y a pas que les icones du menu administrateur qui peuvent être prisent en compte, mais toutes les icones au travers de toutes les pages Coppermine, aussi bien pour les administrateurs que pour les autres visiteurs pour être relevée ou pas avec une petite icone représentant l'action qui peut être faite ou qui a été faite.

Un thème Coppermine peut contenir son propre jeu d'icones. S'il ne contient pas son propre jeu d'icones, c'est le jeu d'icones par défaut se trouvant dans le répertoire ./votre_repertoire_coppermine/images/icons/ qui est utilisé. Si un thème contient son propre jeu d'icones, les icones se trouvent dans le répertoire ./votre_repertoire_coppermine/themes/votre_theme_personnalise_theme/images/icons/. Vous devez explicitement spécifier qu'un thème personnalisé à son propre jeu d'icones. Pour cela, vous devez ouvrir ./votre_repertoire_coppermine/themes/votre_theme_personnalise_theme/theme.php avec un éditeur de texte et ajouter/dé-commenter une ligne comme ceci:

define('THEME_HAS_MENU_ICONS',16);

Si une telle ligne est présente, Coppermine cherchera les icones de menu dans ./votre_repertoire_coppermine/themes/votre_theme_personnalise_theme/images/icons/. Si une telle ligne n'est pas présente, le jeu d'icones par défaut sera utilisé. Le code que vous devez ajouter doit se trouver dans une ligne propre. Elle défini une constante et remplis celle ci avec la valeur "16", qui sera la valeur pour la taille (en pixels) du côté de l'icone de menu.

La fonction icones de menu est relativement nouvelle, donc, avec la publication initile de cpg1.5.x, il n'y aura vraissemblablement pas beaucoup de choix dans les icones de menu. Mais cela peut changer et le nombre de jeux d'icones augmenter. Il est recommandé de vérifier sur le site officile de temps en temps pour de nouveaux jeux d'icones.

Barre Graphique de Navigation

Un thème Coppermine peut contenir un jeu d'images utilisé pour la barre de navigation. La barre de navigation graphique est la ligne au dessus des images intermédiaires qui permets aux visiteurs de passer à l'image précédente, à la suivante d'un album et (si disponible) d'afficher ou de cacher la section des informations de l'image, et permets aux utilisateurs de poster des commentaire et de lancer le diaporama. Si le thème n'a pas de fichiers graphiques propres pour la barre graphique de navigation, c'est le jeu d'images par défaut se trouvant dans le répertoire ./votre_repertoire_coppermine/images/navbar/ qui est utilisé. Si un thème contient un jeu d'images pour la barre de navigation, ces images se trouvent dans le dossier ./votre_repertoire_coppermine/themes/votre_theme_personnalise_theme/images/navbar/. Vous devez spécifier explicitement que votre thème personnalisé a son propre jeu d'images. Pour cela, vous devez ouvrir ./votre_repertoire_coppermine/themes/votre_theme_personnalise/theme.php avec un éditeur de texte et ajouter/dé-commenter une ligne comme ceci:

define('THEME_HAS_NAVBAR_GRAPHICS',1);

Si une telle ligne est présente , Coppermine va cherche les images pour la barre de navigation dans ./votre_repertoire_coppermine/themes/votre_theme_personnalise/images/navbar/. Si une telle ligne n'existe pas, les images par défaut de Coppermine seront utilisées. Le code que vous devez ajouter doit être placé dans une ligne propre. Elle défini une connstante et la remplie avec la valeur "1", valeur qui n'a pas de signification particulière.

Si vous décidez de créer vos propre images pour la barre de navigation de votre thème personnalisé, les images suivantes doivent se trouver dans le répertoire de votre barre de navigation personnalisée (./votre repertoire_coppermine/themes/votre_theme_personnalise/images/navbar/):

  • ecard.png
  • info.png
  • next.png
  • prev.png
  • report.png
  • slideshow.png
  • thumbnails.png

Images de notation

Un thème Coppermine peut contenir des images pour la notation. La possibilité de permettre aux visiteurs d'évaluer les images n'est pas lié à la présence des images de notation - c'est un paramètre qui dépend des permissions des groupes de visiteurs dans (→ Panneau de controle des Groupes) et des propriétés des albums (→ Propriétés de l'Album). Si un thème n'a pas d'images de notation propre, c'est le jeu d'images par défaut qui se trouve dans le dossier ./votre_repertoire_coppermine/images/. Si un thème a ses propres images de notation, elles se trouvent dans le dossier ./votre_repertoire_coppermine/themes/votre_theme_personnalise/images/. Vous devez explicitement définir qu'un thème personnalisé a ses propres images de notation. Pour cela, vous devez ouvrir ./votre_repertoire_coppermine/themes/votre_theme_personnalise/theme.php avec un éditeur de texte et ajouter/dé-commenter une ligne comme ceci:

define('THEME_HAS_RATING_GRAPHICS',1);

Le code que vous devez ajouter doit être placé dans une ligne propre. Elle défini une connstante et la remplie avec la valeur "1", valeur qui n'a pas de signification particulière..

Si défini, les fichiers suivants doivent se trouver dans le répertoire images du thème:

  • rate_empty.png
  • rate_full.png
  • rate_new.png
  • rating.png

Images du négatif de film

Un thème Coppermine peut contenir un jeu d'images utilisées pour composer le négatif de film. Si un thème n'a pas ses propres images de négatif de film, ce sont les images par défaut qui se trouvent dans le dossier ./votre_repertoire_coppermine/images/ qui sont utilisées. Si un thème a son propre jeu d'images pour le négatif de film, les images se trouvent dans le dossier ./votre_repertoire_coppermine/themes/votre_theme_personnalise/images/. Par défaut, l'image utilisée à pour nom tile.gif.

Vous devez explicitement spécifier qu'un thème a son propre jeu d'images pour le négatif de film. Pour celà, vous devez ouvrir ./votre_repertoire_coppermine/themes/votre_theme_personnalise/theme.php avec un éditeur de texte et ajouter/dé-commenter une ligne comme ceci:

define('THEME_HAS_FILM_STRIP_GRAPHIC',1);

Le code que vous devez ajouter doit être placé dans une ligne propre. Elle défini une connstante et la remplie avec la valeur "1", valeur qui n'a pas de signification particulière..
