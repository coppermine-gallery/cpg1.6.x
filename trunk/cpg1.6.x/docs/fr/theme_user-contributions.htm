Thèmes: Contribution des utilisateurs

Coppermine est livré avec quelques thèmes pré-établis, afin de donner à l'utilisateur un point de départ. Dans le sous-forum "cpg1.5 theme contributions",(forum anglophone) vous trouverez d'autres thèmes que vous pourrez télécharger et utiliser pour votre galerie Coppermine.

Prévisualisation des contributions de thèmes d'utilisateurs

Vous pouvez visualiser les thèmes dont nous avons connaissance sur le site de démonstration de cpg1.5.x - Utilisez la liste déroulante en haut de la page pour naviger au travers des thèmes et voir la démo de Coppermine avec l'habillage de chaque thème. De cette manière, vous pouvez voir à quoi ressemble chaque page de Coppermine (index, page des vignettes, images intermédiaires) pour chaque thème.

Si vous voulez juste avoir un rapide tour d'horizon, il y à des copies d'écran de la page d'index de la plupart des thèmes dans la galerie de démonstration dans l'album "User-contributed themes". Pour un tour rapide, regardez le diaporama de cet album. Nénamoins, les albums ne sont pas toujours à jour et ne contiennent pas toujours les contributions les plus récentes, les copies d'écran ne sont mises à jour que de temps en temps.

Télécharger des contributions de thèmes d'utilisateurs

Vous pouvez télécharger toutes les contributions de thèmes dont nou savons connaissance depuis notre référenciel de téléchargement sur sourceforge.net

Par concéquent: si un thème n'existe pas dans la section de téléchargement ni sur la page de démnstration, c'est que nous ne l'avons pas, il n'est donc pas nécessaire de demander pour ce thème en particulier.

Installer des thèmes provenant de conributions d'utilisateurs

Les thèmes sont généralement dans une archive (un fichier .zip). Pour utiliser un thème particulier, téléchargez le et décompressez le fichier zip sur votre disque dur, en gardant la structure de répertoires contenue dans l'archive. Puis utilisez votre client FTP pour palcer le répertoire entier du thème sur votre serveur (dans le sous-répertoire "themes" de votre installation Coppermine).

Admettons que vous ayez téléchargé un pack thème nommé cpg1.4.x_theme_foobar.zip et que votre galerie Coppermine se trouve à l'adresse http://votre_site.com/votre_repertoire_coppermine/. Décompactez le fichier compressé sur votre ordinateur - vous devriez trouver maintenant sur votre ordinateur un répertoire nommé "foobar". Téléchargez ce répertoire (en utilisant une application FTP) sur votre serveur dans le dossier correspondant à http://votre_site.com/votre_repertoire_coppermine/themes/

En concéquence, vous devriez avoir la structure suivant sur votre serveur: http://votre_site.com/votre_repertoire_coppermine/themes/foobar/.

Dans la prochaine étape, testez votre nouveau thème en ajoutant le paramètre theme=foobar dans la barre d'adresse de votr enavigateur (dans l'exemple ci-dessus, vou sdevrez aller à http://votre_site.com/votre_repertoire_coppermine/?theme=foobar). Assurez vous que tout fonctionne correctement. Si c'est le cas, mettez le nouveau thème (qui n'est actuellement visible que par vous administrateur, puisque vous en avez forcé l'affichage en ajoutant le paramètre particulier dans l'adresse URL) comme thème par défaut de votre galerie Coppermine (c.a.d rendez le thème visible à tous vos visiteurs) en allant dans la configuration de Coppermine, et séléctionnez votre nouveau thème "foobar" dans la liste déroulante sous "Paramètres des Thèmes" > "Thème" et appliquez le changement en cliquant sur "Sauvegardez la nouvelle configuration" au bas de la page de configuration. Enfin vérifiez vos réglages en réinitialisant votre choix individuel de thème - allez à http://votre_site.com/votre_repertoire_coppermine/?theme=xxx.

Copyrights des thèmes provenant de contributions d'utilisateurs

Coppermine lui-même est publié sous licence GNU GPL, ce qui signifie basiquement que vous ête sautorisé à l'utiliser sur vos pages web (ausi bien pour un usage commercial que non-commercial). Celà s'applique aussi pour les thèmes livrés avec le pack Coppermine.

Néanmoins, les thèmes provenant de contributions d'utilisateurs peuvent avoir une autre licence différente. Si c'est le cas, le pack du thème que vous pourrez télécharger contiendra un fichier lisez.moi qui expliquera les détails du copyright et les conditions d'utilisation de ce thème en particulier.

Certains thèmes par exemple, obligent à laisser intact le lien vers le site de l'auteur. D'autres stipulent que leur thèms ne peuvent être utilisés que pour des sites non-commerciaux. Nous vous demandons instamment de réspecter la licence des thèmes provenant de contributions d'utilisateurs - merci de lir eles fichier lisez.moi corresondant.

Demande d'aide pour les thèmes provenant de contributions d'utilisateurs

La plupart des contributeurs de thèmes ouvrent un sujet d'annonce séparé sur le sous-forum "cpg1.5 theme contributions". Si vous avez un problème particulier avec un de ces thèmes, asurez vous d'avoir vérifié les sujets d'annonces correspondants sur ce forum. Il y à une fine ligne entre une demande d'aide, une remarque valide pour un thème et le fait de submerger un sujet de demandes particulières. Si vous n'êtes pas sur que votre question doivent se trouver dans le post d'annonce, il est préférable que vous ouvriez votr epropre sujet sur le forum. Votre post devra contenir un lien vers votre galerie ainsi qu'un lien vers le post d'annonce concernant le thème.

Il n'y a aucune garantie que l'auteur d'un thème veuille assurer le support de celui-ci. Ceci dit, vous devez comprendre qu'il n'y a qu'une aide limitée concernant les thèmes provenant de contributions d'utilisateurs.

Contribuer avec votre propre thème

Vos contributions en thèmes sont toujours les bienvneues. Les utilisateurs "normaux" ne peuvent pas poster dans le sous-forum "cpg1.5 theme contributions" afin d'éviter d'être submergés de posts non valides. C'est pourquoi, si vou ssouhaitez contribuer avec votre thème, ouvrez un nouveau sujet sur le forum "cpg1.5 themes/skins/templates". Zippez votre thème et attachez le à votre post (en utilisant les "Additional options (options supplémentaires)" lors de la composition de votr emessage). Une prévisualisation (c.a.d un lien vers votr egalerie utilisant ce thème) est ausi la bienvenue. Un modérateur déplacera votre sujet sur le sous-forum si il est valide.

Même si ce n'est pas obligatoire, il est conseillé de clairement indiquer la licence de votre thème. Si vou sne le faites pas, nous considérerons que votre thème est publié sous licence GNU GPL (la même que Coppermine). Si vous n'avez pas construit votre thème complêtement, mais que vous avez utilisé la base d'un autre thème, regardez sa licence avant tout.

Il est tout à fait accepté de demander de laisser le lien vers le site de l'auteur du thème, cela peut être une motivation pour vous de proposer votre votre thème à la communauté: votre site gagnera en popularité auprès des moteurs de recherche si votre thème devient populaire.
