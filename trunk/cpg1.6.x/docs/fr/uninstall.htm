Désinstallation

Si vous n'avez plus besoin de votre galerie Coppermine et que vous voulez la désinstaller, vous devrez faire deux choses:

  • Effacer les tables de la base de donnée

    Utilisez un outils comme phpMyAdmin pour effcaer toutes les tables Coppermine de la base de donnée (regardez la copie d'écran). Vou spouvez déterminer quelles tables sont à effacer en regardant le préfixe des tables: toutes les tables Coppermine utilisent le préfixe que vous avez défini lors de l'installation de la galerie. Si vous n'êtes pas sur de ce que vous faites, faites une sauvegarde de la base de donnée avant d'effacer définitivement les tables. Par défaut, le préfixe est cpg15x_ (mais il peut être différent si vous avez choisi un autre préfixe lors de l'installation). En utilisant phpMyAdmin, sélectionnez toutes le tables Coppermine, puis allez au bas de la page et choisissez "drop" (Supprimer) du menu déroulant "with selected" (Pour la sélection). confirmez dans la boite de dialogue de sécurité par "yes"(Oui).
    Alternativement (si vous n'avez pas phpMyAdmin de disponible), créez un fichier PHP (en utilisant un éditeur de texte comme notepad.exe) avec ce contenu: 
    <?php
// ajoutez les données de connection mysql ici/add the mysql connection stuff here

DROP TABLE `cpg15x_albums`, `cpg15x_banned`, `cpg15x_bridge`, `cpg15x_categories`, 
`cpg15x_comments`, `cpg15x_config`, `cpg15x_dict`, `cpg15x_ecards`, `cpg15x_exif`, 
`cpg15x_favpics`, `cpg15x_filetypes`, `cpg15x_hit_stats`, `cpg15x_pictures`, `cpg15x_plugins`, 
`cpg15x_sessions`, `cpg15x_temp_data`, `cpg15x_temp_messages`, `cpg15x_usergroups`, 
`cpg15x_users`, `cpg15x_votes`, `cpg15x_vote_stats`;
?>
    sauvegardez le fichier sous "droptables.php" et placez le sur votre serveur (en utilisant votr eapplication FTP) dans le répertoire de votre galerie Coppermine. Puis lancez ce fichier en allant depuis votre navigateur à l'adresse http://votresite.com/votre_repertoire_coppermine/droptables.php
  • Effacez les fichiers
    Utilisez votre application FTP pour effacer les dossiers et les fichiers du répertoire dans lequel vous aviez installé Coppermine.
