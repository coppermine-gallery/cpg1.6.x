Utilisation de l’assistant de publication web, Windows XP Publisher avec Coppermine

Si vous utilisez Windows XP, vous pouvez utiliser son assistant intégré de publication Web pour télécharger vos photos dans votre galerie.

Une fois que vous avez correctement installé le script sur votre serveur, appelez le fichier xp_publish.php depuis votre navigateur web (http://votre_site.fr/votre_dossier_coppermine/xp_publish.php).

Le script affiche des informations sur la manière de faire l'installation du côté client et comment utiliser l'assistant. Fondamentalement, vous aurez besoin de télécharger un petit fichier créé par le script, qui doit être chargé dans votre base de registre Windows.

Si vous souhaitez permettre à vos utilisateurs d'utiliser l’assistant de publication Web de Windows XP, il est conseillé de le lancer en affichant un lien vers le fichier quelque part sur votre page.

Requis

  • XP Publisher est un produit Microsoft. Il ne fonctionne que sur Windows XP / Vista. Il ne fonctionnera que si le client utilise le navigateur de Microsoft "Internet Explorer". Coppermine fournit simplement un fichier pour utiliser l'interface d’utilisateur de XP Publisher. Il n'y a rien qui peut être fait pour contourner cela. Si vous n'êtes pas satisfait de la façon dont fonctionne XP Publisher ou si vous voulez des fonctionnalités supplémentaires ou différentes, vous devrez vous adressez à Microsoft, et non aux développeurs de Coppermine.
  • Comme indiqué plus haut, le fichier de registre a besoin de créer une clé dans la base de registre de l'utilisateur. Il doit y avoir un nom pour cette clé. C'est pourquoi vous devez avoir spécifié un nom à votre galerie et une description de la galerie dans la configuration de Coppermine.

Installation

Avant de pouvoir utiliser l’assistant de publication Web XP Publisher, il faut connaître l'adresse de votre galerie.
  • Ouvrez votre navigateur et tapez la commande suivante dans la barre d'adresse, remplacez "votre_site.fr" et "votre_dossier_coppermine" avec les informations de l’adresse de votre propre galerie : http://votre_site.fr/votre_dossier_coppermine/xp_publish.php
  • Au "Téléchargement de fichier - Avertissement de sécurité", cliquez sur «Enregistrer» pour enregistrer le fichier cpg_###.reg sur votre bureau. (# # # représente un horodatage à 10-chiffres.)
  • L'explorateur de Windows s'ouvre
  • Sélectionnez bureau dans la colonne de gauche
  • Double-cliquez sur le nom du fichier que vous venez d'enregistrer
  • Dans l’éditeur de registre, à la question "Etes-vous sûr ..." cliquez sur boîte de dialogue "Oui"
La procédure ci-dessus ne doit être effectuée qu'une seule fois par les utilisateurs qui souhaitent utiliser l’assistant de publication XP-Publisher.

Télécharger des photos

Le processus de transfert de photos est une suite de simple boîte de dialogue. La description du processus est plus longue que son exécution.

  • Ouvrez l'explorateur de Windows et recherchez le répertoire contenant vos photos, sélectionnez en une ou plusieurs pour les télécharger.
  • Si dans l'explorateur de Windows la colonne de gauche n’est pas intitulé "Gestion d’image", cliquez sur le «X» dans le coin supérieur droit du panneau pour le fermer et laisser voir le panneau de Gestion d'image.
  • Sélectionnez "Publier xxx sur le web" de "fichier et dossier de tâches" sur le panneau de gauche. "Xxxx" peut vouloir dire "ce fichier, ces fichiers ou ce dossier" selon ce qui est mis en évidence sur le panneau à droite.
  • Au message de bienvenue de l’assistant de publication Web , cliquez sur l'écran, "Suivant".
  • Vous pouvez modifier votre choix si nécessaire dans la fenêtre montrant l’aperçu. Cliquez sur "Suivant" lorsque vous êtes prêt.
  • Dans la fenêtre, mettre en évidence le nom de votre galerie et cliquez sur "suivant".
  • Entrez votre nom d'utilisateur et votre mot de passe de Coppermine pour accéder à votre galerie et cliquez sur "Suivant".
  • Dès la fenêtre de "Bienvenue votre identifiant", vous avez la possibilité de télécharger votre photo (s) dans l'un des albums existants ou à partir d'un nouvel album. Cliquez sur "Suivant" lorsque vous êtes prêt.
  • Cliquez sur "Suivant" dans la fenêtre de confirmation pour télécharger
  • Sélectionnez la taille des images pour les images téléchargées. Si vous n'êtes pas sûr du format d'image accepté, vérifiez auprès de votre administrateur de Galerie. Cliquez sur "Suivant" lorsque vous êtes prêt.
  • Vous êtes a présent sur la dernière boite de dialogue et devez donner un choix pour l'ouverture de la galerie, lorsque le transfert est terminé.
  • Cliquez sur "Terminer" terminer l’assistant de publication web XP Publisher et entrer dans la galerie pour vérifier le contenu du nouvel album.
  • Afin d'éviter que les galeries ne puissent être envahies par les images téléchargées par le biais de l'assistant, seuls les administrateurs de galeries et les utilisateurs qui peuvent avoir leurs propres albums peuvent utiliser cette fonctionnalité.
  • Une fois le téléchargement commencé, l'assistant ne peut pas afficher de message d'erreur renvoyé par le script, de sorte que vous ne pouvez savoir si le téléchargement a échoué ou réussi jusqu’au moment où vous vérifiez votre galerie.
  • Si le téléchargement échoue, l'administrateur devrait autoriser "Activer le mode débogage" dans la configuration de Coppermine et essayer avec une seule image et vérifier les messages d'erreur dans le fichier du journal qui se trouve dans le répertoire racine de Coppermine sur le serveur.

Structure des fichiers

Beaucoup d’administrateurs de galerie préfèrent avoir un dossier physique de la structure des fichiers au sein de leur dossier albums qui représente la structure logique catégorie/albums de la galerie. L’utilisation d’XP Publisher, ne peut conserver la structure des dossiers/fichiers: les fichiers téléchargés en utilisant l'assistant de publication XP Publisher vont aller dans un dossier albums qui suit une règle particulière de nommage: l'Assistant crée des dossiers nommés "wpw-YYYYMMDD" (par exemple "wpw-20081026") et télécharge les fichiers dans ce dossier. Il n'est pas possible de contourner ou de modifier ce comportement, mais après tout l’homme n’a pas à le faire car : Coppermine garde une trace des fichiers dans sa base de données.

Suppression/Désinstallation

Si vous souhaitez supprimer l’entrée de l’assistant de publication Web XP Publisher créer pour votre galerie (par exemple parce que vous avez déménagé vers un nouveau domaine), vous devez modifier le Registre manuellement. Faites attention lorsque vous le faite - jouer avec le registre sans savoir ce que vous faites peut endommager votre système d'exploitation.

  • Cliquez "Démarrer" - "Exécuter..."
  • Taper "regedit" (sans les guillemets) et appuyez sur Entrée
  • Accédez à HKEY_CURRENT_USER \ Software \ Microsoft \ Windows \ CurrentVersion \ Explorer \ PublishingWizard \ PublishingWizard \ Providers \
  • Sélectionnez l'entrée sur le côté gauche de l'écran qui correspond à votre installation de Coppermine
  • Appuyez sur la touche "supprimer" de votre clavier
  • Cliquez sur "Oui" la boîte de dialogue de confirmation
  • Fermez l'éditeur de registre
