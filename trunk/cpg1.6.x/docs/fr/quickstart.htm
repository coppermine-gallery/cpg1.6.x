Question: Mon serveur supporte t'il Coppermine ?
Réponse: Lisez la section sur le minimum requis.

Question: Ou dois-je mettre tous ces fichiers ?
Réponse: Vous pouvez les mettre dans le répertoire racine (non recommandé) ou les placer dans un répertoire Coppermine de votre site web. Si vous n'êtes pas sur de savoir exactement dans quel répertoire le faire, demandez à votre hébergeur dans quel répertoire vous devez télécharger vos fichiers html et autres fichiers de script. Nous vous recommandons de télécharger le répertoire cpgXXXX (et tous les fichiers qu'il contient) directement sur votre serveur dans le répertoire recommandé par votre hébergeur, et de le renommer comme vous le souhaitez après le téléchargement complet. Comparez le contenu du pack de Coppermine avec les fichiers téléchargés. Assurez-vous que tous les fichiers ont été téléchargés correctement.

Question: Ok, j'ai téléchargé tout sur www.monsite.com/cpgXXX. Et maintenant ?
Réponse: Vous devez changer les permissions d'un certain nombre de répertoires avant de faire quoi que ce soit d'autre. Utilisez votre client FTP ou le Panneau de contrôle de votre site pour changer les droits pour les répertoires: "include", "albums", "albums/userpics", et "albums/edit" à 777. Vous devrez aussi créer un nouveau sous-répertoire dans le répertoire albums dans lequel vous placerez vos images/fichiers par FTP (nommez le "uploads" ou de la manière que vous souhaitez - simplement n'y mettez pas d'espaces ou d'autres caractères spéciaux.) et paramétrez aussi les permissions à '777'. Sur certains serveurs, un chmod à '777' ne fonctionnera pas. Dans ce cas, utilisez '755'. (Perdu ? vous ne savez pas ce qu'est le CHMOD ? Vous avez été prévenus, devons-nous vous dire autre chose ?)

Question: J'ai changé les permissions. Quelle est la prochaine étape ?
Réponse: Vous devez créer une base de données MySQL. Utilisez le panneau de contrôle de votre site ou un outil comme PhpMyAdmin et créez la base de données. Notez le nom de la base de données, le nom d'utilisateur et le mot de passe - vous en aurez besoin lors de l'installation de Coppermine. Si vous avez déjà une base de données utilisée par un forum ou une autre application, cela fonctionnera aussi. Vous pouvez aussi utiliser cette base de données pour Coppermine. Si vous avez un forum et que vous planifiez de le connecter à Coppermine, vous devrez de toute manière partager la base de données.

Question: Mes fichiers sont en place, les autorisations sont définies, et j'ai mes renseignements concernant ma base de données MySQL. Quelle est la prochaine étape ?
Réponse: Lisez la... (Oops, pardon). Maintenant vous devez lancer l'installation de Coppermine. Ouvrez votre navigateur web et entrez http://www.votresite.com/votre_repertoire_coppermine/install.php puis entrez les informations demandées. Si vous n'avez jamais utilisé ImageMagik avant, laissez cette ligne vide pendant le processus d'installation. Vous pourrez toujours ajouter ces informations par la suite, une fois l'installation terminée et lorsque votre galerie Coppermine fonctionnera correctement.

Question: Ha! Ha! Je l'ai eu! C'est installé et ça marche! Et maintenant ?
Réponse: 1) Effacez ce sourire de votre visage. 2) Lisez la documentation, encore -- une deuxième fois. 3) Vous devrez regarder la configuration de Coppermine pour être sûr que la galerie est paramétrée comme vous le souhaitez. Identifiez-vous comme administrateur si vous n'êtes pas encore identifié. Cliquez sur le bouton Configuration. Il y a plusieurs choses auxquelles vous devez faire attention:

  • Paramètres généraux: Vous devrez entrer le nom de votre site, et votre adresse courriel ainsi que votre zone horaire GMT.
  • Paramétrage des langues, thèmes & jeux de caractères: Choisissez votre langue par défaut.
  • Paramétrages des fichiers et vignettes: Définissez la taille maximale pour le téléchargement des fichiers que vous autorisez (en Kilobytes). Le paramétrage par défaut est 1024kb. Assurez-vous que cette taille maximale est plus petite ou égale à celle autorisée par votre hébergeur. Tant que vous y êtes, vous devriez aussi définir la "Largeur ou hauteur maximale pour le téléchargement des images/vidéos (pixels) téléchargés" - Cela empêchera vos visiteurs de télécharger des images plus grandes que ce que vous souhaitez. Si vous désirez que Coppermine redimensionne automatiquement les images plus grandes que ce que vous avez défini dans la configuration, paramétrez "Redimensionne automatiquement les images plus grandes que la largeur ou hauteur maximale" sur soit "Oui: Tout le Monde" ou "Oui: Utilisateur uniquement". "Tout le monde" signifie tout le monde, y compris lorsque l'administrateur (vous) téléchargez les images par lot, et "Utilisateurs" ne s'applique qu'aux membres de votre galerie lorsqu'ils téléchargent des images.
  • Paramètre avancés pour les fichiers et les vignettes: Assurez-vous que vous avez sélectionné le bon programme de manipulation d'images. Si vous utilisez GD, Vous devriez sélectionner GD2 s’il est disponible sur votre serveur. Si vous souhaitez utiliser ImageMagik, vous devrez entrer le chemin vers le programme (assurez vous que le chemin se termine par un slash de fin "/"). Voulez vous permettre aux membres de créer des albums privés ? Si oui, assurez-vous d'avoir paramétré "Les utilisateurs peuvent avoir des albums privés" sur 'Oui'.
  • Paramétrage des utilisateurs: Si vous souhaitez laisser les visiteurs s'enregistrer et télécharger des images, assurez-vous que "Permettre les nouvelles inscriptions" est paramétré à "Oui".
  • Vous voulez également jeter un œil dans les réglages de groupes. Coppermine est livré avec trois groupes par défaut: les administrateurs, les utilisateurs inscrits, les utilisateurs anonymes. vous contrôlez les autorisations d'envoyer des e-cartes, télécharger des fichiers, etc C'est également là que vous pouvez définir la quantité maximale autorisée de l'espace disque pour chaque groupe (quota d'espace disque). Il est positionné à 1024kb par défaut.

Question: Je pense que tout est paramétré correctement. Je peux télécharger les images maintenant ?
Réponse: Non. Vous devez d'abord créer un album pour les y placer. Dans la même ligne que le lien pour la Configuration vous verrez un bouton ou un lien "Albums". Cliquez dessus et vous serez amené à un écran qui vous permettra de créer un album. Cliquez sur "Nouvel Album", puis renommé le comme vous le souhaitez. Cliquez sur "Appliquer les changements" et voilà, vous avez un album maintenant.

Question: J'ai créé un album, alors je peux maintenant télécharger mes images ?
Réponse: Oui. En tant qu'administrateur, vous devriez utiliser un client FTP pour télécharger vos images votre serveur, puis faites un ajout par lot pour les enregistre dans votre base de données Coppermine. Au début de ce guide, nous vous demandions de créer un sous-répertoire dans le répertoire "Albums" justement dans cet objectif. Admettons que vous l'ayez nommé "uploads". Placez vos images par FTP dans votre répertoire albums/uploads. Une fois terminé, cliquez sur le bouton FTP. Sélectionnez "uploads" comme répertoire. Vous avez maintenant devant vous une liste de tous les fichiers que vous venez de télécharger. Par défaut, Coppermine sélectionne toutes les images. Sélectionnez l'album dans lequel vous voulez placer les images puis cliquez sur "envoyer". Vous avez maintenant une galerie en ligne avec vos images visibles! Incroyable, Non ?

Question: Maintenant, je peux ajouter un titre à une image ?
Réponse: Oui vous pouvez. Sur l'écran principal de Coppermine, en étant identifié comme administrateur, vous verrez une liste d'options à côté de l'album. Si vous cliquez sur "modifier les photos", vous pourrez ajouter toutes sortes d'informations pour chaque image.

Question: Je peux effacer l'image originale maintenant Non? Après tout, elles sont dans la base de données, c'est çà ?
Réponse: Faux! Coppermine ne place pas réellement les images dans la base de données. Il utilise la base de données uniquement pour se rappeler ou les images sont stockées sur le serveur, quel album et quelle catégorie lui sont attachés ainsi que les informations relatives à l'image. Si vous effacez les images originales, vous aurez une série d'erreurs. Si vous voulez effacer les images originales, cherchez dans la documentation à quoi servent les outils Administrateurs et ce pourquoi ils sont faits. Vous ne devriez jamais effacer quoi que ce soit en utilisant les outils de votre hébergeur ou de votre client FTP si cela peut être fait directement par Coppermine. En le faisant, vous aurez des erreurs sur votre serveur et dans votre base de données MySQL.

Vous exécuter à présent Coppermine avec succès sur votre site ! Maintenant, prenez quelques minutes pour lire la documentation afin que vous sachiez comment administrer votre galerie. Coppermine est très puissant et peut faire beaucoup plus que ce que vous avez sans doute pensé possible au départ.

Expérimenter le avec juste quelques images, catégories et albums dans un premier temps. Réglez-le exactement comme vous le voulez avant d'ajouter un tas de photos. La plupart passe un bon moment ! Ah oui, et n'oubliez pas, RTFM!
