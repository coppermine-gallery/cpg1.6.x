BBCode

Utilisez le BBCode pour insérer des liens et des mises en forme spéciales dans les différents champs de description.

Bulletin Board Code ou BBCode est un langage de balisage léger utilisé dans de nombreux forums et applications web semblables.

But

Les balises disponibles sont généralement indiquées par des crochets entourant le mot-clé, elles sont analysées par le système avant d'être traduites en un langage de balisage que les navigateurs web comprennent - le plus souvent HTML ou XHTML.

En d'autres termes: les balises BBCode, vous permettent d'ajouter une mise en forme de texte sans donner aux visiteurs à l'accès à toutes les commandes HTML (ce qui serait un risque de sécurité élevé et donc n'est pas une option).

Les balises BBCode disponibles

Coppermine accepte les BBCodes suivants (les mêmes BBCodes qui sont utilisés par phpBB, SMF et de nombreuses autres applications BBS) dans les images, les champs de description d'album ainsi que dans les zones de commentaires.

  • [b]Texte en gras[/b] => Texte en gras
  • [u]Texte souligné[/u] => Texte souligné
  • [i]Texte en italique[/i] => Texte en italique
  • [s]Texte barré[/s] => Texte barré
  • [color=red]Texte colorié en rouge[/color] => Texte colorié en rouge
  • [url=http://unsite.com/]Lien hypertexte[/url] => Lien hypertexte
  • [url=http://votresite.com/]Lien hypertexte[/url] => Lien hypertexte
  • [email]utilisateur@domain.com[/email] => utilisateur@domain.com
  • [img]http://documentation.coppermine-gallery.net/images/bullet.png[/img] =>

Dangers

Dans cpg1.4.x, l'équipe de développement de Coppermine a dû désactiver le traitement des balises BBCode [url] et [img] avec la version de maintenance de cpg1.4.21 en raison des dangers potentiels d'une attaque Cross-Site Request Forgery (CSRF qualifier). Dans cpg1.5.x, ce danger a été traité par l'ajout d'une vérification de la valeur d'un champ spécial dans le traitement de tous les formulaires. Cette initiative vise à faire en sorte que le lien maquillé ne puisse pas causer une action réservée à l'administrateur. Cependant, il reste néanmoins un risque dans l'autorisation de l'utilisation des balises mentionnées.

Plus de contrôle

Il y a une possibilité plus puissante et graduelle de contrôle des BBCode disponible sous forme de plugin séparé qui n'est pas livré directement avec Coppermine. Si vous avez besoin de plus de balise BBCode ou si vous devez explicitement activer ou désactiver certaines balises, jetez un œil au plugin de contrôle BBCode.
