Coppermine Photo Gallery v1.5.x: Documentatie en Handleiding

Inhoudstafel

Gekende problemen

De volgende lichte kronkels en kleine problemen zijn gekend en zullen hopelijk in een van de volgende versies opgelost worden.

  • De pijlen die voor navigatie gebruikt worden om van de ene naar de andere afbeelding te gaan, worden in een verkeerde volgorde getoond bij talen die van rechts naar links geschreven worden
  • Trefwoordenlijst geeft een onjuist resultaat voor Aziatische tekens
  • Browser bug: IE toont de 100%-breedte waardes onjuist
  • De nieuwe interface voor inbatch-toevoegingen werkt niet bij alle server setups. Gebruikers die hiermee problemen hebben, kunnen hiervoor de klassieke interface gebruiken.
  • Paginatabs in gebruikersbeheer gedragen zich niet hetzelfde als andere tabs
  • Herschalen van gif/png met transparante achtergrond behoudt de alpha info niet
  • Verkeerde karakterset in e-mails die verzonden worden door Coppermine
  • Thumbnail-afmetingen boven de 120 px veroorzaken dat de editvelden buiten hun grenzen gaan
  • Geen album thumbnails voor albums die alleen afbeeldingen bevatten die gelinkt zijn door trefwoorden
  • Knippen/draaien werkt niet met GD1
  • Draaien veroorzaakt een vervormde afbeelding bij grote bestanden
  • <object><param></param></object> constructie voor multimedia bestanden valideren niet maar zijn nodig voor neerwaartse compatibiliteit
  • EXIF en IPTC informatie gaat verloren bij verdraaien van afbeeldingen
  • Discrepantie tussen Coppermine intern gebruik van utf-8 en IPTC (momenteel alleen ondersteuning voor iso-8859-1). Bijgevolg worden de niet latin karakters als accentletters of Umlauts vervormd
  • Album wordt aangemaakt bij gebruik van XP publisher ondanks een uploadfout
  • Geen categorie-thumbnail voor gebruikersgalerijen en categorieën die geen albums bevatten met afbeeldingen in maar alleen subcategorieën
  • Filmstrip kan van hoogte wisselen tijdens het navigeren als de thumbafmeting op 'breedte' ingesteld is
  • Flash uploader (swfupload) werkt niet als de galerij zich in een paswoord beveiligde map bevindt
  • Sommige ronde-hoekeffecten van het curve-thema werken niet in alle browsers maar zijn eigen aan bepaalde browsers. De browsers die de ronde hoeken niet kunnen weergeven tonen gewone rechte hoeken.
  • Album moderator feature not clearly documented and therefore disabled (thread)
  • Category thumbnails appear larger than expected if thumb aspect is set to "exact" for the thumbnail setting "Use dimension" (thread)
  • If you allow guests to upload files to your gallery, they won't be able to add file details. This feature has been disabled as it's not possible to identify the user without fail (see thread). Additionally it may attract spammers to add textual content to your gallery. If you want to enable that feature anyway, find Coppermine's configuration table in the database and change the value of allow_guests_enter_file_details to '1'.
